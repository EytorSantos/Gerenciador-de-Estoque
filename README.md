# Sistema de Gestão de Estoque para Farmácias

Projeto desenvolvido como critério avaliativo para a disciplina de **Engenharia de Software I (CC0116)**. O objetivo deste software é automatizar o controle de entrada, saída e validade de medicamentos, visando otimizar a gestão operacional de uma farmácia.

* **Controle de Validade:** Alertas para produtos próximos ao vencimento.
