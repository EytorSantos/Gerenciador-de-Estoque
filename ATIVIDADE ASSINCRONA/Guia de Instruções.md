# INSTRUÇÕES DE VISUALIZAÇÃO DO PROJETO

Para uma visualização correta do relatório e do diagrama de classes (CASO queira baixa a ATIVIDADE ASSINCRONA), observe as orientações abaixo:

## 1. ESTRUTURA DOS ARQUIVOS:
   O arquivo 'document.md' (Markdown) está configurado para exibir a imagem 
   'diagrama_classes_profissional.png'. Portanto, ambos devem permanecer 
   na mesma pasta dentro do repositório para que o link da imagem funcione.

## 2. FERRAMENTAS RECOMENDADAS:
   Para visualizar o arquivo 'document.md' com a formatação correta (incluindo 
   o renderizador de Markdown e a imagem), recomenda-se o uso de:
   
   - VS Code: Abra o arquivo e clique no ícone de "Open Preview to the Side" 
     (ícone de um documento com uma lupa no canto superior direito).
   - GitHub: Ao clicar no arquivo 'document.md' dentro do repositório, o 
     GitHub renderiza automaticamente o Markdown e exibe o diagrama.
   - Editores especializados: Obsidian, Typora ou extensões de Markdown para navegadores.

Caso o diagrama não apareça, verifique se o nome do arquivo de imagem 
no código do Markdown está exatamente igual ao nome do arquivo enviado.
